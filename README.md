# Vantage HR Prototype

## Overview

Vantage HR Prototype is a Human Resource Management (HRM) and Recruitment Platform prototype designed to streamline workforce management, employee administration, recruitment workflows, onboarding processes, payroll management, and organizational reporting.

The project demonstrates modern web development practices and serves as a foundation for building a scalable HR management solution.

## Installation

Clone the repository:

```bash
git clone https://github.com/MONALISA2003-CMD/vantage-hr-prototype.git
```

Navigate to the project directory:

```bash
cd Vantage-HR-Management-System-
```

Open the project:

```bash
index.html
```
